## Clone this repo

```
git clone git@github.com:fandredev/react-quiz.git
```

## Go to directory

```
 cd react-quiz
```

## Install dependencies

```
 yarn install
```

## Run API Server

```
 yarn run server
```

## Run WEB server

```
 yarn run dev
```
